# 📌 Sobre o Projeto

Este projeto foi desenvolvido utilizando **Blazor**, com o objetivo de criar uma aplicação de currículo digital personalizada e mais profissional.

A atividade propõe a atualização visual e estrutural da aplicação **MeuCurriculo Blazor**, adicionando informações profissionais, melhorias no layout e personalização da interface.

# ▶️ Guia de Execução

## Pré-requisitos

Antes de executar o projeto, é necessário ter instalado:

- [.NET SDK](https://dotnet.microsoft.com/download)

---

## Executando o Projeto

### 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/una-blazor-lista14.git
```

---

### 2. Acesse a pasta do projeto

```bash
cd una-blazor-lista14
```

---

### 3. Execute a aplicação

```bash
dotnet run
```

---

### 4. Abra no navegador

Após executar o comando, acesse:

```txt
http://localhost:5000
```

ou a porta exibida no terminal.
